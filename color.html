Helpers de Color

Bootstrap 5 ofrece clases utilitarias para aplicar colores predefinidos tanto a texto como a fondos, facilitando la personalización visual sin escribir CSS adicional.

Texto

Texto azul (primary) - Este texto utiliza la clase text-primary para aplicar un color azul predefinido.

Texto verde (success) - Este texto utiliza la clase text-success para aplicar un color verde predefinido.

Texto rojo (danger) - Este texto utiliza la clase text-danger para aplicar un color rojo predefinido.

Texto amarillo (warning) - Este texto utiliza la clase text-warning para aplicar un color amarillo predefinido.

Texto azul claro (info) - Este texto utiliza la clase text-info para aplicar un color azul claro predefinido.

Texto atenuado - Este texto utiliza la clase text-muted para aplicar un color gris atenuado.

Texto oscuro - Este texto utiliza la clase text-dark para aplicar un color gris oscuro.

Fondo

Fondo verde - Este fondo utiliza la clase bg-success para aplicar un color verde predefinido.

Fondo gris - Este fondo utiliza la clase bg-secondary para aplicar un color gris predefinido.
Fondo claro - Este fondo utiliza la clase bg-light para aplicar un color claro predefinido.
