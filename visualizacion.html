Helpers de Visualizacion

Estas clases controlan cómo y cuándo se muestra un elemento.

d-none: Este elemento no es visible.

No visible

d-block: Este elemento es visible como un bloque.

Visible como bloque

d-inline: Este elemento es visible en línea.

Visible en linea

d-flex: Este elemento es un contenedor flex.

Contenedor flex
Responsive

d-md-none d-lg-block: Este elemento está oculto en pantallas medianas (md) y visible en pantallas grandes (lg).

Oculto en md, visible en lg
