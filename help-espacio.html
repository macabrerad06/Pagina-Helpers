Helpers de Espaciado en Bootstrap

Bootstrap proporciona clases utilitarias para aplicar márgenes y rellenos fácilmente.

¿Cómo funcionan?

La sintaxis general es:

.m{lado}-{tamaño} para margen (margin)
.p{lado}-{tamaño} para relleno (padding)

Donde:

  • m = margin, p = padding
  • lado: t (top), b (bottom), s (start), e (end), x (horizontal), y (vertical), nada (todos)
  • tamaño: de 0 a 5 (o auto para margenes)

Ejemplo 1: Margen

Agregamos margen inferior con .mb-4 (margin bottom 4):

Caja con .mb-4
Caja siguiente

<div class="bg-warning p-3 mb-4">Caja con .mb-4</div>
<div class="bg-secondary text-white p-3">Caja siguiente</div>

Ejemplo 2: Padding

Agregamos relleno horizontal con .px-5 (padding left/right 5):

Caja con .px-5

<div class="bg-info text-white px-5 py-3">Caja con .px-5</div>

Ejemplo 3: Margen automático

Utilizamos .mx-auto para centrar horizontalmente:

Caja centrada con .mx-auto

<div class="bg-danger text-white w-50 mx-auto p-3">Caja centrada con .mx-auto</div>
